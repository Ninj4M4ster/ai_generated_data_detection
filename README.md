# AI Generated Data Detection
This project aims to explore performance of current detectors for ai generated context.

A dataset of human and ai generated texts was picked (silentone0725/ai-human-text-detection-v1) and two models from Hugging Face were used for detection: roberta-base-openai-detector and gpt2. Additionally, GPTZero was used as a third tool for detection.

Roberta model was used in its original form - output is 0 or 1 indicating whether the text is human or ai generated. GPT2 was used to compute perplexity of the text, which was then converted to a binary label using a threshold. GPTZero does not have an API, so it was used by manually inputting the texts and recording the outputs. That is why there are significantly less results for GPTZero.

Finally, the same process was repeated for 10 new texts generated by ChatGPT with different prompts, which were then paraphrased to look more human-like. The results were compared to see if the detectors perform differently on these new texts.
