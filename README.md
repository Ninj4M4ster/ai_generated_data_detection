# AI Generated Data Detection
This project aims to explore performance of current detectors for ai generated context and propose a new method for this task.

## Venv creation

```bash
python3.12 -m venv .venv && source .venv/bin/activate && python -m pip install --upgrade pip setuptools wheel && pip install -r requirements.txt
```

## Running code

The code is in the [notebook](project/notebook.ipynb), all the cells can be run from for example VS Code "Run All", or a similar button in different IDEs.
